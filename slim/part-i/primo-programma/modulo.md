#### L'operatore modulo

Oltre alle classiche quattro operazioni,
tutti i linguaggi di programmazione mettono a disposizione una quinta operazione:
il resto della divisione, detto anche *modulo*.

Apri l'interprete Python e prova:

```
$ python
```

```py
>>> 7 % 4
3
>>> 10 % 2
0
```
