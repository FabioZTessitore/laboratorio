#### Il ciclo while

Un tipico ciclo `while` scritto in C assume la forma:

```c
/* contatore */
int i = 0;

while (i < 3) {
  puts("Hello");

  /* aggiorna il contatore! */
  i++;
}
```

In Python non cambia la sostanza:

```py
# contatore
i = 0

while i < 3:
  print "Hello"

  # aggiorna
  i += 1
```
