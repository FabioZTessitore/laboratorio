**Lista**<br>
Scrivi un programma Python che, data una lista di valori qualsiasi,
ne stampi gli elementi uno per riga.

Esempio di output:

```
Ho memorizzato la lista seguente:
[1, 2, 3, 'hello', 4.2, (0, 0)]

Gli elementi della lista, uno per riga, sono:
1
2
3
'hello'
4.2
(0, 0)
```
